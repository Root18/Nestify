# Support

## How to Get Help

If you need help with Nestify, here are the best ways to get support:

### 1. Documentation

Start by reading the [README](README.md) for an overview of features, setup instructions, and usage.

### 2. GitHub Issues

If you encounter a bug or unexpected behavior:
- Search [existing issues](https://github.com/Root18/Nestify/issues) to see if your problem has already been reported.
- If not, [open a new issue](https://github.com/Root18/Nestify/issues/new/choose) with as much detail as possible.

### 3. GitHub Discussions

For general questions, ideas, or community conversations, use [GitHub Discussions](https://github.com/Root18/Nestify/discussions) (if enabled on the repository).

### 4. Feature Requests

Have an idea to improve Nestify? Open a [feature request](https://github.com/Root18/Nestify/issues/new?template=feature_request.md) and describe what you would like to see.

## What This Project Does NOT Provide

- **Guaranteed response times** — This is a community-driven open source project maintained by volunteers.
- **Paid or enterprise support** — There is no commercial support offering at this time.

## Contributing

If you want to help improve Nestify, check out the [Contributing Guide](CONTRIBUTING.md) to get started.
