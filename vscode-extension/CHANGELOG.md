# Change Log

## 0.1.0

Initial release.

- **Nest under...** — pick a parent for one or more selected files from the Explorer context menu.
- **Unnest** — remove exact-name nesting entries for the selected files (workspace and user settings).
- **Auto-nest** — apply the Nestify naming rules (C# interfaces, markdown documentation, minified/bundled JS) as glob patterns.
- **Remove auto-nest rules** — remove exactly the rules Auto-nest added.
- **Toggle file nesting** — flip `explorer.fileNesting.enabled`.
